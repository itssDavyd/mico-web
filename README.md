# 🎙️ MICO - Meeting Intelligence & Content Organizer

![SvelteKit](https://img.shields.io/badge/SvelteKit-blue?style=for-the-badge&logo=svelte&logoColor=white) ![Svelte](https://img.shields.io/badge/Svelte-4A4A55?style=for-the-badge&logo=svelte&logoColor=FF3E00) ![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white) ![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white) ![pnpm](https://img.shields.io/badge/pnpm-F69220?style=for-the-badge&logo=pnpm&logoColor=white) ![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)

MICO es una aplicación web de grabadora inteligente que utiliza inteligencia artificial para transcribir automáticamente grabaciones de audio a formato PDF. La aplicación permite gestionar las grabaciones, visualizarlas en un listado y descargarlas o eliminarlas según sea necesario.

## Características

- 🎙️ Grabación de audio directamente desde el navegador
- 🤖 Transcripción automática mediante IA
- 📄 Generación de PDFs con las transcripciones
- 📋 Listado y gestión de grabaciones
- ⬇️ Descarga de transcripciones en PDF
- 🗑️ Eliminación de grabaciones

## Stack Tecnológico

- **Framework**: SvelteKit
- **Gestor de paquetes**: pnpm
- **Frontend**: Svelte

## Instalación

Instala las dependencias del proyecto:

```sh
pnpm install
```

## Desarrollo

Inicia el servidor de desarrollo:

```sh
pnpm dev

# o abre la aplicación automáticamente en el navegador
pnpm dev -- --open
```

## Compilación

Para crear una versión de producción:

```sh
pnpm build
```

Puedes previsualizar la compilación de producción con `pnpm preview`.
